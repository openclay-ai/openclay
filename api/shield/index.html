    API Reference: Core Shield

    This section provides the auto-generated documentation for the Shield core API.

    promptshield.shields.Shield

    Configurable security shield with PyTorch-style API.

    Compose security components declaratively instead of fixed levels.

    Examples:

    Full customization

    shield = Shield( patterns=True, models=["xgboost"], canary=True, rate_limiting=True, pii_detection=True )

    Use presets

    shield = Shield.fast() # <1ms shield = Shield.balanced() # ~1ms shield = Shield.secure() # ~5ms

    Customize preset

    shield = Shield.balanced(pii_detection=True)

    balanced(**overrides) classmethod

    Balanced preset: Patterns + canary, ~1ms

    Best for: Most production use cases

    fast(**overrides) classmethod

    Fast preset: Pattern-only, <0.5ms

    Best for: High-throughput APIs, agent-to-agent

    from_config(config_path, **overrides) classmethod

    Create a Shield from a YAML/JSON config file.

    This enables centralized, code-free security policy management. Security teams can update policies without touching application code.
